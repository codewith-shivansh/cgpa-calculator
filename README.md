# Namdapha CGPA Calculator

A client-side CGPA calculator for IIT Madras BS Degree students.

## Features
- Supports DSAI and ES degrees
- Current & Future CGPA calculation
- Fully client-side (No backend)
- Clean UI with Tailwind CSS
- Built using Next.js & TypeScript

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS

## How to Run Locally

1. Install dependencies:
   npm install

2. Run development server:
   npm run dev

3. Open in browser:
   http://localhost:3000
